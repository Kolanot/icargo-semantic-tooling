rdfs:comment
The XML tag used to render an element. The values of this property must be valid XML tag strings or URI resources that can be abbreviated by an existing namespace prefix. For example, a tag would be mapped into a class with sxml:element="process"^^xsd:string. A tag would be mapped into a class with sxml:element=my:process (which is often identical to the URI of the class itself).
