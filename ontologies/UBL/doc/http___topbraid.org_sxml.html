An ontology describing XML documents (elements and their hierarchy). XML Documents have a root element. Attributes are represented as values of datatype properties that have a value for sxml:attribute. Future versions may also support object properties to resolve IDREF attributes. Elements are represented as instances of classes that have an sxml:element annotation.
